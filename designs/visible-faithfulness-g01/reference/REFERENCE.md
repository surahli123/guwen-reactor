## 设计系统（从截图提取）
- **中心 = 沉浸式青绿山水手卷地图**（主角，非侧景）。绢本米底 `#e8dab9`，石青 `#2f5b6d`/石绿 `#3f7458`（发暗矿物色，非霓虹），云雾 + 天空大留白。→ **应放公有领域《千里江山图》crop 作真底**（Wikimedia PD，base64 内嵌保持单文件离线）。
- **城池签牌**：小米色竖排文字牌，钉在山水上；**选中 = 深松绿底 + 米字**。
- **灯火**：签牌旁琥珀发光点（原义:诗篇多少 → 我们:**已核实之光**）。box-shadow 叠层暖光。
- **右侧册页面板**：选中即从右滑入的做旧书页——大字标题 + 元数据计数 + 名句引文框 + 列表（每行右侧一枚**朱红小印章**）。古籍/刻本质感。→ **这是我们的"阅读 + 考据"面。**
- **竖排书法标题**左上；**印章**左下；**右上控件签牌**。
- 字：标题行楷/魏碑（跨机需内嵌子集 LXGW），正文宋体，列表小宋。

## 映射到我们的 G01 管寧割席（单则叙事，非地理地图）
