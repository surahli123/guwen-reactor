# Guwen Reactor Demo

This is the official static demo for 看得见的忠实 / Visible Faithfulness: a multi-article hub for *Shishuo Xinyu* that makes source-grounding and verification status visible while staying fully offline.

Open `index.html` in any browser. No server, network, API key, or build step is needed.

The working prototype remains at `designs/visible-faithfulness-g01/prototypes/proto-d-multiarticle-hub.html`.
